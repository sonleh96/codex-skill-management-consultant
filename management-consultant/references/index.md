# Management Consultant Reference Index

Use this file before loading detailed reference files. Pick the 1-3 most relevant references for the user's task, then read only those files.

## Selection Rules

1. Match the user's task to a problem type, industry, method, or deliverable.
2. Prefer the narrowest applicable reference over broad references.
3. For current market, company, regulatory, pricing, benchmark, or trend facts, use web search with citations instead of relying only on static references.
4. If a task spans multiple domains, load one core problem-solving reference plus one domain-specific or method-specific reference first.
